# Comunicação Não-Violenta (CNV)

A Comunicação Não-Violenta é uma ferramenta poderosa para criar conexões e evitar mal-entendidos em comunidades online baseadas em texto.

## Os 4 Pilares da CNV

1. **Observação**: Descreva o fato sem julgamento.
   - *Exemplo*: "O build falhou após o commit X" em vez de "Você quebrou o build de novo".
2. **Sentimento**: Expresse como se sente (opcional em contextos técnicos, mas válido).
   - *Exemplo*: "Fico preocupado com a estabilidade..."
3. **Necessidade**: Identifique o que é necessário para o projeto ou para você.
   - *Exemplo*: "...precisamos garantir que os testes passem."
4. **Pedido**: Faça um pedido claro e realizável.
   - *Exemplo*: "Poderia verificar os logs de erro e corrigir?"

## Dicas para Textos

- Evite ironia e sarcasmo, que são difíceis de interpretar por escrito.
- Use emojis para suavizar o tom quando apropriado .
- Releia sua mensagem antes de enviar, pensando em como o outro receberá.
