# Ambientes de Desktop (DE)

Um Ambiente de Desktop (Desktop Environment - DE) é um conjunto de softwares
que oferece uma interface gráfica completa e integrada para o usuário interagir
com o sistema operacional.

Ao contrário de apenas um gerenciador de janelas, uma DE geralmente inclui:

* **Gerenciador de Janelas (WM):** Para controlar como as janelas são
  desenhadas e movidas.
* **Gerenciador de Arquivos:** Para navegar pelas pastas e arquivos.
* **Painéis e Menus:** Barras de tarefas, menus de aplicativos e bandejas do
  sistema.
* **Utilitários de Configuração:** Ferramentas para ajustar rede, som, vídeo
  e aparência.
* **Aplicativos Básicos:** Editor de texto, terminal, visualizador de
  imagens, etc.

## Exemplos Populares

### GNOME

O GNOME é um dos ambientes mais populares e padrão em muitas distribuições como
Fedora e Ubuntu. Possui um fluxo de trabalho único, focado em atividades e
pesquisa, com uma interface moderna e minimalista.

### KDE Plasma

Conhecido por sua extrema personalização e semelhança (por padrão) com o layout
clássico do Windows. É leve, rápido e permite que o usuário altere praticamente
qualquer aspecto da interface.

### XFCE

Um ambiente leve, estável e tradicional. Ideal para computadores mais antigos
ou para quem prefere uma interface simples e direta, sem muitos efeitos visuais
pesados.
