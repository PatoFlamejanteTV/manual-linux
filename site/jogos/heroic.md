# Heroic Games Launcher

O **Heroic Games Launcher** é um iniciador de jogos de código aberto para Linux,
Windows e macOS. Ele serve como uma alternativa nativa para a Epic Games Store e
GOG Galaxy.

## Recursos

- **Epic Games Store:** Permite baixar, instalar e jogar jogos da sua biblioteca
  da Epic.
- **GOG:** Integração completa com a GOG.com, incluindo suporte a salvamento na
  nuvem para jogos compatíveis.
- **Amazon Prime Gaming:** Suporte para instalar jogos resgatados no Prime
  Gaming.
- **Wine/Proton:** Gerencia automaticamente versões do Wine e Proton (incluindo
  versões GE-Proton) para rodar os jogos.

## Por que usar?

As lojas oficiais da Epic e GOG não possuem clientes nativos para Linux. O
Heroic preenche essa lacuna oferecendo uma interface limpa, rápida e que
facilita muito a configuração do Wine para esses jogos.

## Instalação

Geralmente distribuído via Flatpak, AppImage ou pacotes nativos (DEB/RPM/AUR).

```bash
# Via Flatpak
flatpak install flathub com.heroicgameslauncher.hgl
```
