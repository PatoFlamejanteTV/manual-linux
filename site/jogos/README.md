# Jogos no Linux

O cenário de jogos no Linux evoluiu drasticamente nos últimos anos. Graças a
camadas de compatibilidade avançadas e suporte de grandes empresas, jogar no
Linux hoje é uma realidade viável e muitas vezes com desempenho comparável ao
Windows.
