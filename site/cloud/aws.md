# AWS

**Amazon Web Services (AWS)** é a plataforma de nuvem mais abrangente e
amplamente adotada do mundo, oferecendo mais de 200 serviços completos de
datacenters em todo o mundo.

## Principais Serviços

- **EC2 (Elastic Compute Cloud)**: Servidores virtuais escaláveis.
- **S3 (Simple Storage Service)**: Armazenamento de objetos escalável.
- **RDS (Relational Database Service)**: Banco de dados relacional gerenciado.
- **Lambda**: Computação serverless (execução de código sem provisionar
  servidores).
