# Manual Básico de Linux

Este repositório serve como um guia introdutório e simplificado sobre o funcionamento do terminal Linux e os conceitos fundamentais do ecossistema.

O objetivo é fornecer explicações claras e diretas para quem está começando a explorar o mundo do software livre.

## Estrutura do Repositório

O conteúdo está organizado nas seguintes pastas:

*   [`/comandos`](./comandos): Uma referência rápida para os comandos mais comuns do dia a dia.
*   [`/conceitos`](./conceitos): Explicações sobre ideias e termos importantes, como distribuições, imagens ISO e ambientes Live.
*   [`/etica`](./etica): Discussões sobre a filosofia e os princípios que norteiam o software livre, como o Projeto GNU e o conceito KISS.

Sinta-se à vontade para explorar os tópicos e usar este material como um ponto de partida para seus estudos.
