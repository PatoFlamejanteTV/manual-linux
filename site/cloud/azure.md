# Azure

**Microsoft Azure** é um conjunto de serviços de nuvem em constante expansão
para ajudar sua organização a enfrentar seus desafios de negócios. É a nuvem da
Microsoft.

## Principais Serviços

- **Virtual Machines**: Máquinas virtuais Linux e Windows.
- **Azure Blob Storage**: Armazenamento de objetos massivamente escalável.
- **Azure SQL Database**: Banco de dados SQL gerenciado e inteligente.
- **Azure Functions**: Solução serverless para executar pequenos pedaços de
  código.
