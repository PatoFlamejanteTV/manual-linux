# man

O comando `man` é o seu melhor amigo no Linux. Ele abre o manual de
instruções de quase qualquer outro comando. Se você está perdido e não sabe
como usar algo, tente dar um `man` nele!

Exemplo de uso:

```bash
$ man ls
```

Isso vai abrir uma página explicando tudo sobre o `ls`. Para sair do manual,
basta apertar a tecla `q`.
