# Processo de Boot

O processo de boot é a sequência de eventos que ocorrem quando o computador é ligado.

## Etapas do processo de boot

1. **BIOS/UEFI**: O firmware da placa-mãe inicializa o hardware.
2. **Bootloader**: O bootloader (como o GRUB) carrega o kernel.
3. **Kernel**: O kernel inicializa o sistema.
4. **Init/Systemd**: O processo init/systemd inicia os serviços do sistema.
