# Gerenciadores de Janelas (WM)

O Gerenciador de Janelas (Window Manager - WM) é o software responsável por
posicionar e gerenciar a aparência das janelas na interface gráfica. Ele pode
ser parte de um Ambiente de Desktop (DE) ou usado sozinho para um sistema mais
leve e personalizado.

## Tipos de WM

### Stacking (Floating)

São os mais comuns (como no Windows ou macOS). As janelas flutuam umas sobre as
outras, podem ser redimensionadas livremente e sobrepostas.

* **Exemplos:** Openbox, KWin (do KDE), Mutter (do GNOME).

### Tiling (Ladrilho)

As janelas não se sobrepõem. Elas são organizadas automaticamente em uma grade
(ladrilhos) para ocupar todo o espaço da tela. É muito focado no uso do
teclado.

* **Exemplos:** i3, Sway, bspwm, Hyprland.

### Dynamic

Permitem alternar entre o modo tiling e o modo stacking conforme a necessidade.

* **Exemplos:** AwesomeWM, DWM.
