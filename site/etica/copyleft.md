# Copyleft

**Copyleft** é um método geral para tornar um programa (ou outra obra) livre e exigir que todas as versões modificadas e estendidas do mesmo também sejam livres. É o oposto de *Copyright* (direitos autorais) no sentido de que usa a lei de direitos autorais para garantir liberdades em vez de restringi-las.

A licença mais famosa que implementa o Copyleft é a **GPL** (GNU General Public License).

## Como funciona?

Se você distribui um software com licença Copyleft (seja o original ou uma versão modificada por você), você deve:

1. Passar as mesmas liberdades para quem recebe a cópia.
2. Disponibilizar o código-fonte.
3. Não impor restrições adicionais.

Isso garante que o software permaneça livre para sempre, impedindo que alguém pegue o código livre, modifique-o e feche-o (tornando-o proprietário).
