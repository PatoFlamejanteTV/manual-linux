# `GNU` / The GNU Project

O conceito GNU/Linux é um sistema operacional que combina o kernel Linux (o gerenciador de recursos principal, praticamente a base da base) com a vasta coleção de ferramentas de software GNU (compilador, shell, utilitários, etc..) para criar um sistema completo e independente.

É uma alternativa colaborativa e orientada pela comunidade a sistemas proprietários como o Windows e MacOS, construída sobre princípios de liberdade e transparência.
