# JavaScript (Node.js) no Linux

Node.js permite rodar JavaScript no lado do servidor.

## Instalação (Via NVM)

A melhor forma é usar o Node Version Manager (nvm):

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
source ~/.bashrc
nvm install --lts
```

## Instalar pacotes

```bash
npm install express
```
