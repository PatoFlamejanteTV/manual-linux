# manual-linux

Um guia pequeno e simples sobre como o terminal do Linux funciona, sem enrolação.

## O que tem aqui?

*   [`/comandos`](./comandos): Uma "colinha" dos comandos mais úteis.
*   [`/conceitos`](./conceitos): Explicações rápidas sobre coisas como "distros", "ISOs", etc.
*   [`/etica`](./etica): Um pouco da filosofia por trás do software livre.

A ideia é ser um ponto de partida. Explore, aprenda e, o mais importante, divirta-se!
