# uname

O **uname** exibe informações sobre o sistema.

## Todas as informações

Kernel, hostname, arquitetura, etc.

```bash
$ uname -a
Linux quack 6.12.63+deb13-amd64 #1 SMP PREEMPT_DYNAMIC Debian 6.12.63-1 (2025-12-30) x86_64 GNU/Linux
```

## Versão do Kernel

```bash
$ uname -r
6.12.63+deb13-amd64
```

## Nome do Kernel

```bash
$ uname -s
Linux
```

## Arquitetura da Máquina

```bash
$ uname -m
x86_64
```
