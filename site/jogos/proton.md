# Proton

**Proton** é uma ferramenta de compatibilidade desenvolvida pela Valve
Corporation que permite que jogos exclusivos para Windows sejam executados em
sistemas operacionais baseados em Linux. O Proton é baseado no Wine e integra
componentes adicionais como DXVK (uma implementação de DirectX sobre Vulkan)
para melhorar o desempenho de jogos 3D.

## Características

- **Integração com Steam:** O Proton é integrado diretamente no cliente Steam
  para Linux.
- **Desempenho:** Graças ao DXVK e VKD3D, muitos jogos rodam com desempenho
  próximo ou até superior ao Windows.
- **Facilidade:** Não requer configuração manual complexa para a maioria dos
  jogos.

## Como ativar na Steam

1. Abra as **Configurações** da Steam.
2. Vá para a aba **Steam Play**.
3. Marque "Ativar o Steam Play para títulos compatíveis".
4. Você também pode marcar "Ativar o Steam Play para todos os outros títulos"
   para tentar rodar jogos que não foram oficialmente verificados.

## ProtonDB

O site [ProtonDB](https://www.protondb.com/) é um recurso essencial onde a
comunidade relata o funcionamento de jogos com o Proton, oferecendo dicas e
parâmetros de inicialização para corrigir problemas.
