# whoami

O comando `whoami` é um comando simples que mostra o nome de usuário do usuário atualmente logado. É útil quando você está em um sistema com vários usuários e precisa saber qual usuário está usando no momento.

## Exemplos

```bash
$ whoami
quack
```

## Argumentos Suportados

Apesar de ser um comando muito simples, o `whoami` aceita os seguintes argumentos:

*   `--help`: Mostra uma mensagem de ajuda com as opções do comando.
*   `--version`: Mostra a versão do `whoami`.
