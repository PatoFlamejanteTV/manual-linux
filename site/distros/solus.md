# Solus

O Solus (anteriormente Evolve OS) é uma distribuição Linux independente, construída do zero (não é baseada em nenhuma outra). Diferente da maioria das distros que tentam servir para servidores, IoT e desktops, o Solus foca **exclusivamente no Desktop Pessoal (Home Computing)**.

## Budgie Desktop

O Solus é o criador e principal mantenedor do ambiente de desktop **Budgie**. O Budgie é desenhado para ser moderno, elegante e simples, utilizando tecnologias do GNOME por baixo, mas com uma interface mais tradicional e leve.

## Gerenciamento de Pacotes (eopkg)

Utiliza o gerenciador de pacotes **eopkg**, que é um fork do antigo gerenciador do Pardus Linux. É focado em fornecer atualizações de forma "curated rolling" (rolling release curada).

Isso significa que o Solus é rolling release (você instala uma vez e atualiza para sempre), mas as atualizações são seguradas e testadas todas as semanas e liberadas geralmente às sextas-feiras ("Sync Friday"), garantindo que você não receba um update quebrado no meio da semana.

## Prós (Vantagens)

1. **Otimização:** Como foca apenas em desktop x86_64, o sistema é muito otimizado desdo o kernel até as aplicações.
2. **Visual:** O Budgie é bonito e funcional "out-of-the-box".
3. **Foco Claro:** Não perde tempo com pacotes de servidor, focando em fazer Steam, navegadores e drivers funcionarem perfeitamente.

## Contras (Desvantagens)

1. **Software Limitado:** O repositório é bem menor que o de outras grandes distros. Se o software não estiver lá e não tiver um Flatpak/Snap/AppImage, é difícil instalar.
2. **Equipe Pequena:** Sendo independente, depende de uma equipe menor de desenvolvedores, o que já causou hiatos no desenvolvimento no passado.

## Para Quem é?

* Usuários domésticos comuns.
* Desenvolvedores que não precisam de ferramentas de servidor exóticas.
* Gamers e criadores de conteúdo que querem um sistema estável e bonito.
