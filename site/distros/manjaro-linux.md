# Manjaro Linux

O Manjaro é uma distribuição baseada no [Arch Linux](arch-linux.md), criada na Áustria, França e Alemanha. Seu objetivo é tornar o poder e a flexibilidade do Arch acessíveis a usuários iniciantes e intermediários, removendo a complexidade da instalação e configuração manual.

O lema é "Enjoy the Simplicity" (Aproveite a Simplicidade).

## Características Principais

O Manjaro oferece os benefícios de ser *Rolling Release* (atualização contínua), mas com uma camada extra de testes. As atualizações chegam primeiro ao Arch, são testadas pela equipe do Manjaro e depois liberadas para os usuários, garantindo maior estabilidade.

Ele vem com instalador gráfico (Calamares), detecção automática de hardware (MHWD) para instalar drivers de vídeo (NVIDIA/AMD) e acesso facilitado ao **AUR (Arch User Repository)**.

## Edições Oficiais

1. **Xfce:** A edição principal (flagship). Rápida, robusta e visualmente integrada com o tema do Manjaro.
2. **KDE Plasma:** Para quem deseja máxima personalização e recursos modernos.
3. **GNOME:** Para quem prefere um fluxo de trabalho moderno e simples.

Existem também muitas edições comunitárias (i3, Sway, Budgie, Cinnamon, etc.).

## Prós (Vantagens)

1. **Acesso ao AUR:** Provavelmente o maior repositório de software do mundo Linux.
2. **Rolling Release Estável:** Software sempre recente, mas com menos risco de quebra que o Arch puro.
3. **Fácil Instalação:** O instalador é simples e cuid de partições e drivers.
4. **Pamac:** Um excelente gerenciador de pacotes gráfico que suporta repositórios oficiais, AUR, Flatal e Snaps.

## Contras (Desvantagens)

1. **Atraso nas Atualizações:** Pacotes de segurança podem demorar um pouco mais para chegar que no Arch, pois passam por uma fase de retenção para testes.
2. **Customizações Excessivas:** Algumas configurações padrão podem interferir no comportamento padrão esperado de pacotes do Arch upstream.
3. **Não é Arch Puro:** A comunidade do Arch Linux geralmente não oferece suporte a usuários do Manjaro devido às diferenças nos repositórios e configurações.

## Para Quem é?

* Usuários que querem software recente (Rolling Release) sem a dor de cabeça de configurar o Arch manualmente.
* Gamers (devido ao fácil acesso a drivers e Steam).
* Quem quer acesso ao vasto repositório AUR com uma interface amigável.
