# clear

O comando `clear` é usado para limpar a tela do terminal. Ele remove todo o texto e comandos anteriores, deixando o terminal com uma aparência limpa. É útil quando a tela está muito cheia e você quer se concentrar em um novo comando.

## Exemplos

```bash
$ # A tela está cheia de texto
$ clear
# A tela agora está limpa
```

## Argumentos Suportados

O `clear` não tem muitos argumentos, mas alguns podem ser úteis:

*   `-x`: Não tenta limpar o histórico de rolagem.
