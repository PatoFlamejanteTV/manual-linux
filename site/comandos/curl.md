# curl

O `curl` é uma ferramenta para transferir dados de ou para um servidor. Na
prática, a gente usa muito para baixar arquivos da internet ou testar
conexões com sites.

Exemplo de uso:

```bash
curl https://google.com
```

Existem outro usos interassantes com essa ferramenta também, eu recomendo
usar o irmão dela, o [`wget`](wget) para baixar arquivos de sites e servidores
remotamente com segurança.
