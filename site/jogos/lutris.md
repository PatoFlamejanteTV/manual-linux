# Lutris

**Lutris** é uma plataforma de preservação de jogos de código aberto para Linux.
Ele ajuda você a instalar e gerenciar seus jogos em uma interface unificada.

## O que ele faz

O Lutris facilita a instalação de jogos de várias fontes, incluindo:

- GOG
- Steam
- Battle.net
- Origin
- Uplay
- Emuladores (RetroArch, Dolphin, etc.)
- Jogos de Windows via Wine

## Scripts de Instalação

A grande força do Lutris são os scripts de instalação mantidos pela comunidade.
Eles configuram automaticamente o ambiente Wine, instalam dependências e aplicam
correções para que o jogo rode da melhor forma possível.

## Instalação

```bash
# Debian/Ubuntu (pode requerer adicionar repositório)
sudo apt install lutris

# Fedora
sudo dnf install lutris

# Arch Linux
sudo pacman -S lutris
```
