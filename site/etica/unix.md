# The Unix Philosophy, `unix`

A Filosofia Unix é um conjunto de princípios de design de software que enfatiza a criação de programas pequenos, simples e que fazem uma única coisa bem feita. Esses programas podem ser combinados de maneiras poderosas para realizar tarefas complexas.

Os princípios centrais da Filosofia Unix são:

1.  **Faça uma coisa e faça-a bem:** Cada programa deve se concentrar em uma única tarefa. Em vez de tentar criar um programa que faça tudo, crie vários programas menores que possam ser combinados.
2.  **Escreva programas que trabalhem juntos:** A saída de um programa deve ser a entrada de outro. Isso permite que você crie pipelines de comandos que podem processar dados de maneiras complexas.
3.  **Escreva programas para lidar com fluxos de texto, pois essa é uma interface universal:** O texto é um formato de dados simples e universal que pode ser processado por qualquer programa. Ao usar texto como entrada e saída, você garante que seus programas possam ser combinados com outros programas de maneiras flexíveis.
