# Compilação

Compilar é traduzir código fonte (humano) para código de máquina (binário).

## O Processo

Geralmente envolve:

1. **Pré-processamento**: Prepara o código.
2. **Compilação**: Transforma em Assembly.
3. **Assembly**: Transforma em código objeto.
4. **Linkagem**: Junta tudo num executável.

## Ferramentas

* **gcc**: O compilador C clássico do GNU.
* **make**: Automatiza a compilação lendo um `Makefile`.
