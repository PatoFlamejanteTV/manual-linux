# Ricing, rice

Ricing é um dos muitos jeitos de se expressar usando o Linux, pelo menos de modo visual.

_Linux ricing_ é a prática de personalizar intensamente a aparência e o comportamento de um ambiente de desktop _Linux_ para torná-lo **único, esteticamente agradável e adaptado às necessidades do usuário;**
Similar a tunar carros, mas focado em ícones, temas, gerenciadores de janelas (como _i3_, _Hyprland_) e fluxos de trabalho, buscando um visual "doce" (do inglês _sweet_, que soa como _rice_) ou simplesmente algo visualmente impressionante, indo muito além das opções padrão de outros sistemas operacionais.

O que envolve o ricing?

 - Personalização Visual:
   - Troca de papéis de parede;
   - Icones;
   - Fontes;
   - Cores e temas para criar uma estética coes
     - (minimalista, retrofuturista, etc.).
 - Gerenciadores de Janelas (WMs):
   - Uso de WMs como i3, Hyprland ou Sway, que organizam janelas automaticamente (tiling) para otimizar o espaço e a produtividade.
 - Widgets e Barras de Status:
   - Adição de barras personalizadas (como Polybar);
   - Widgets (como Conky) para exibir informações do sistema.
 - Fluxos de Trabalho:
   - Configuração de atalhos de teclado;
   - Scripts para automatizar tarefas e tornar a navegação mais eficiente
