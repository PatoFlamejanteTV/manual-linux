# Zorin OS

O Zorin OS é uma distribuição baseada no [Ubuntu](ubuntu.md) projetada especificamente para ser a porta de entrada mais fácil para usuários vindos do Windows e macOS. Seu principal objetivo é fazer com que o computador seja poderoso e fácil de usar, permitindo que você use o sistema do jeito que já conhece.

## Zorin Appearance

O grande diferencial do Zorin é a ferramenta **Zorin Appearance**. Com um clique, você pode mudar o layout da área de trabalho para imitar:

* Windows 10/11 (Menu iniciar estilo lista/tiles)
* Windows 7 (Menu clássico)
* macOS (Dock centralizada - disponível na versão Pro)
* Unity/Ubuntu (Barra lateral - disponível na versão Pro)

## Edições

1. **Zorin OS Core:** A versão padrão gratuita, completa para a maioria dos usuários.
2. **Zorin OS Lite:** Otimizada para computadores antigos (até 15 anos atrás), usando o desktop Xfce modificado.
3. **Zorin OS Pro (Paga):** Inclui layouts extras (macOS, Windows 11 Classic, Ubuntu), softwares criativos pré-instalados e suporte técnico.

## Prós (Vantagens)

1. **Familiaridade:** A melhor distro para quem não quer aprender uma nova interface. A transição do Windows é quase imperceptível.
2. **Suporte a Apps Windows:** Vem com Wine e PlayOnLinux facilitados, detectando quando você tenta rodar um `.exe` e sugerindo a instalação correta.
3. **Polimento:** Visual profissional, efeitos suaves e tema muito bem acabado.
4. **Zorin Connect:** Integração com Android (baseado no KDE Connect) para ver notificações e transferir arquivos.

## Contras (Desvantagens)

1. **Recursos Pagos:** Alguns layouts de interface (como o do macOS) estão bloqueados atrás da versão Pro paga.
2. **Base LTS:** Às vezes demora para ter as versões mais recentes do Kernel ou drivers de vídeo, pois foca na base estável do Ubuntu.

## Para Quem é?

* Refugiados do Windows que querem mudar para Linux com zero curva de aprendizado.
* Empresas e escolas que precisam de uma interface familiar.
* Quem quer reviver PCs antigos com a versão Lite sem sacrificar a beleza.
