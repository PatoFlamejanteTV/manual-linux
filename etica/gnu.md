# O Projeto `GNU`

O sistema que a gente chama de "Linux" é, na verdade, uma combinação de duas coisas muito importantes:

1.  **O kernel Linux:** É o chefão, o "gerenciador de recursos principal", a base da base. Ele que conversa com o hardware do seu computador.
2.  **As ferramentas GNU:** É todo o resto que faz o sistema ser útil. O compilador, o shell (aquela tela preta que a gente digita comandos), os utilitários (`ls`, `cat`, `mv`, etc.)... uma coleção gigante de software.

Juntando os dois, você tem o **GNU/Linux**, um sistema operacional completo e independente.

É uma alternativa colaborativa, feita pela comunidade, para sistemas proprietários como o Windows e o macOS, e tudo construído sobre os princípios de liberdade e transparência.
