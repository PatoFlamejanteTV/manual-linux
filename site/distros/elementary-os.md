# elementary OS

O elementary OS é uma distribuição baseada no [Ubuntu LTS](ubuntu.md), famosa por seu foco obsessivo em design, interface de usuário e consistência visual. Ele é frequentemente chamado de "o macOS do mundo Linux" devido à sua estética e filosofia.

O sistema utiliza seu próprio ambiente de desktop chamado **Pantheon**.

## Filosofia de Design e The AppCenter

O elementary OS segue diretrizes de interface humana (HIG) muito estritas. Os desenvolvedores criaram um ecossistema onde os aplicativos se parecem e se comportam de maneira idêntica.

A **AppCenter** é a loja de aplicativos do sistema, focada no modelo "pague o quanto quiser". Ela encoraja desenvolvedores a criar apps nativos de alta qualidade especificamente para o elementary OS.

## Diferenciais

* **Pantheon Desktop:** Interface limpa com um dock inferior (Plank), painel superior transparente e menu de aplicativos simples.
* **Gestos Multitoque:** Excelente suporte para touchpads, similar ao Mac.
* **Privacidade:** Foco em não coletar dados do usuário.
* **Curadoria:** Os aplicativos sugeridos são revisados para garantir segurança, privacidade e consistência visual.

## Prós (Vantagens)

1. **Beleza:** Indiscutivelmente uma das distros mais bonitas "out-of-the-box".
2. **Simplicidade:** Remove muitas opções de configuração para que o usuário não "quebre" o sistema. Foco no essencial.
3. **Estabilidade:** Baseado no Ubuntu LTS, garantindo uma base sólida.
4. **Coesão:** Todo o sistema parece uma peça única de software, não uma colcha de retalhos.

## Contras (Desvantagens)

1. **Restritivo:** Difícil de personalizar. Não há tema escuro global fácil (em versões antigas) ou suporte a ícones na área de trabalho por padrão.
2. **AppCenter Limitada:** A loja foca em apps nativos; instalar apps populares comuns (Chrome, VS Code) requer baixar o `.deb` ou usar Flatpak via terminal inicialmente (embora o suporte a Flatpak tenha melhorado).
3. **Minimalismo Exagerado:** Algumas ferramentas básicas podem faltar na instalação padrão.

## Para Quem é?

* Usuários vindos do macOS.
* Pessoas que valorizam design e estética acima de custosmização.
* Iniciantes que querem um sistema que não os sobrecarregue com opções.
