# `pkg`

No mundo Linux, a palavra "`pkg`" pode significar algumas coisas diferentes, dependendo do contexto.

*   **Pacotes de Software:** De forma geral, "`pkg`" é a abreviação de "package" (pacote). Um pacote é um arquivo que contém um programa completo (executáveis, bibliotecas, arquivos de configuração, etc.), pronto para ser instalado ou removido de forma fácil.

*   **`pkg-config`:** É uma ferramenta para desenvolvedores. Ela ajuda a encontrar informações sobre as bibliotecas instaladas no sistema, o que facilita na hora de compilar um programa que depende delas.

*   **Comando Específico:** Em alguns sistemas, `pkg` é o nome do próprio gerenciador de pacotes. Por exemplo:
    *   No **Termux** (um emulador de terminal para Android), `pkg` é um comando que simplifica o uso do `apt`.
    *   No **FreeBSD** (um sistema operacional parente do Linux), `pkg` é o comando padrão para gerenciar pacotes.
    *   No **Arch Linux**, o gerenciador de pacotes é o `pacman`, mas muita gente se refere aos pacotes como "pkgs".
