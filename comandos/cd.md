# `cd`

Permite que você mude de pasta (ou "diretório").

```bash
# Estando na minha pasta de usuário, vou entrar na pasta 'Documentos'
$ cd Documentos

# Agora, se eu quiser voltar para a pasta anterior, uso '..'
$ cd ..
```

Alguns atalhos úteis:

*   `cd ~`: Leva você direto para a sua pasta de usuário.
*   `cd -`: Leva você para o último diretório em que esteve.
