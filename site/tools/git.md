# Git

Git é o sistema de controle de versão distribuído mais popular do mundo, criado por Linus Torvalds (o mesmo criador do Linux). Ele é essencial para desenvolvedores de software.

O Git permite rastrear mudanças em arquivos, reverter para versões anteriores e trabalhar em colaboração com outras pessoas.

## Comandos Básicos

Iniciar um repositório:
```bash
$ git init
```

Verificar o estado dos arquivos:
```bash
$ git status
```

Adicionar arquivos para serem "commitados":
```bash
$ git add arquivo.txt
```

Salvar as mudanças (commit):
```bash
$ git commit -m "Minha primeira mudança"
```
