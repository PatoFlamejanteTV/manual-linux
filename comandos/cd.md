# `cd`

O comando `cd` (abreviação de "change directory") é usado para navegar entre diretórios (pastas) no terminal.

A sintaxe básica é:
`cd [CAMINHO_DO_DIRETORIO]`

### Exemplos

**Navegar para um subdiretório:**
Para entrar em um diretório chamado `documentos`, use:
```bash
$ cd documentos
```

**Retornar ao diretório anterior:**
Para voltar ao diretório em que você estava antes, use:
```bash
$ cd -
```

**Navegar para o diretório pai:**
Para subir um nível na hierarquia de pastas, use `..`:
```bash
$ cd ..
```

**Ir para o diretório home:**
Executar `cd` sem argumentos leva diretamente ao seu diretório home:
```bash
$ cd
```
