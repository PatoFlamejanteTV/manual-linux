# Vim

Vim (Vi IMproved) é um editor de texto altamente configurável construído para
criar e alterar qualquer tipo de texto de forma muito eficiente. É incluído como
"vi" na maioria dos sistemas UNIX.

## Modos Básicos

O Vim opera em diferentes modos:

* **Modo Normal**: Para navegação e manipulação de texto. Pressione `Esc` para
  voltar a este modo.
* **Modo de Inserção**: Para digitar texto. Pressione `i` para entrar.
* **Modo Visual**: Para selecionar texto. Pressione `v` para entrar.
* **Modo de Comando**: Para salvar, sair, etc. Pressione `:` no Modo Normal.

## Comandos Comuns

Abrir um arquivo:

```bash
$ vim arquivo.txt
```

Salvar e sair:

```bash
:wq
```

Sair sem salvar:

```bash
:q!
```
